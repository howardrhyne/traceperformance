# Trace Performance Center

Golf instruction website for Carolina Trace Country Club featuring PGA Professionals Ed Walls and Jeffrey Taylor.

## Files

- `index.html` - Main landing page
- `trace-performance-center.html` - Detailed instructor page  
- `page1.html` - Jeffrey Taylor's putting instruction page

## Setup

This is a static website that can be hosted on GitHub Pages or any web server.

Note: Image files are not included in the repository due to size constraints. Please add your own images to the directory.
